Unix job control

https://www.thegeekdiary.com/understanding-the-job-control-commands-in-linux-bg-fg-and-ctrlz/

  • $binary & send binary to background (create a job)
  • jobs list jobs
  • fg % $job_num bring job to foreground
  • bg % $job_num send job to backgrouns
  • CTRL+Z - stop job and send it to background

Tar archive

  • tar -czvf foo.tar dir Create archive (c)create,(z)compress,(v)verbose progress, (f)output file name
  • tar -xvf foo.tar Extract archive
  • Add -C to change working dir

Libvirt virsh

virsh domifaddr $DOMAIN_NAME    # returns IP address information
virsh console $DOMAIN_NAME`     # login via console, exist wtih CTRL+SHIFT+]

Process grep (pgrep)

pgrep -f firefox # -f makes it match against whole command line not just the name
