Safe python execution via Docker and Django Web UI

projects
So I decided to explore the idea of running python code in docker. Always wanted to try Django web framework so this was my chance J. What I ended up doing is tiny Web page where you can enter python code which gets executed inside docker container and resulting python console output displayed to user. Sure this is just a proof of concept, but something close to this could be used to safely execute custom code provided by client in some real product.
